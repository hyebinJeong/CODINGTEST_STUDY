## Coding Test Study 목적
알고리즘 문제 해결 능력을 기르고,<br>
문제 접근 과정과 코드 구현에 대한 사고력을 향상시키는 것을 목표로 한다.

## 📖 Study Method
매주 동일한 문제를 각자 풀고, 풀이를 공유하며 접근 방식과 구현 차이를 비교한다.

- `WeekXX` : 주차별 문제
- 각 주차 폴더 안에는 **이름별 풀이 코드**가 있다.

## 📝 Study Rules
- 각자는 `WeekXX/본인이름/` 폴더에 업로드한다.
- 각자는 **본인 이름 폴더만** 수정한다.
- 파일명은 문제 이름으로 통일한다.
